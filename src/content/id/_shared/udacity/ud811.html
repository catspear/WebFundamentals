Mengenal Progressive Web Apps

Dalam kursus ini Anda akan mulai mengerjakan Aplikasi Web Progresif pertama anda - aplikasi web yang dapat memanfaatkan banyak fitur yang dinikmati aplikasi native. Anda juga akan mendapatkan lebih banyak pengalaman dalam membuat aplikasi web yang bekerja secara offline yang menggunakan Service Workers. Akhirnya, Anda akan membuat aplikasi Anda yang dapat diinstal ke layar awal pengguna dengan file Manifest Web App.

Ini Adalah Penawaran Kursus Gratis Udacity
